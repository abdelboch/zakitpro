# Drawy Whiteboard Spec — AI-Assisted Driver Update Approval Review for Desktop Engineers

## Goal
One whiteboard social visual for a safer enterprise driver approval workflow.

## Format
- 1:1 square (1080x1080)
- hand-drawn whiteboard style

## Core message
"AI can speed up driver review. Engineers still own the risk."

## Layout
- Header: "AI + Driver Approval Review"
- Main flow (5 steps):
  1) Gather driver packet
  2) Compare old vs new metadata
  3) Ask AI to argue against approval
  4) Pilot by model family
  5) Write rollback notes first
- Right-side warning box:
  - Newer does not always mean safer
  - Dock, Wi-Fi, graphics, audio = tighter pilots
- Bottom badges:
  - Fewer surprise regressions
  - Better pilot coverage
  - Cleaner rollback plans
