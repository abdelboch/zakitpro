# Drawy Whiteboard Spec — AI + Windows Autopatch Deployment Rings

## Goal
Create one whiteboard-style social visual that explains how desktop engineers can use AI to validate Windows Autopatch deployment rings safely.

## Format
- 1:1 square (1080x1080)
- hand-drawn whiteboard style
- dark marker on white board

## Core message
"Use AI to compare Autopatch ring configs faster, but make engineers validate rollout order, targeting, and risk before production."

## Layout
- Header: "AI + Windows Autopatch Rings"
- Center flow with 5 steps:
  1) Export ring config
  2) Normalize settings
  3) Compare with AI
  4) Flag rollout risks
  5) Validate on pilot devices
- Right-side warning box:
  - AI can summarize differences
  - AI should not approve production rollout
- Bottom badges:
  - Faster reviews
  - Safer rollout checks
  - Less config drift
