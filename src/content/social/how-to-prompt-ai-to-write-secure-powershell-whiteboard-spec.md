# Drawy Whiteboard Spec — How to prompt AI to write secure PowerShell

## Goal
Summarize secure AI prompting workflow for PowerShell generation.

## Format
- 1080x1080
- whiteboard sketchnote
- clear hierarchy for mobile readability

## Core message
"Secure AI PowerShell starts with constraints, not code."

## Layout
- Header: "Secure AI PowerShell Prompting"
- Main flow (6 steps):
  1) Define scope/blast radius
  2) Add security constraints
  3) Require threat analysis
  4) Require negative tests
  5) Human review + analyzer
  6) Pilot with rollback
- Side "Never" box:
  - no Invoke-Expression
  - no hard-coded secrets
  - no destructive defaults
- Bottom outcomes:
  - fewer risky scripts
  - better auditability
  - safer production changes
