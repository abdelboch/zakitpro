# Drawy Whiteboard Spec — Using AI to generate Intune detection rules

## Goal
One whiteboard social visual for enterprise detection rule workflow.

## Format
- 1:1 square (1080x1080)
- hand-drawn whiteboard style

## Core message
"AI can draft Intune detection logic fast; engineers make it reliable through matrix testing."

## Layout
- Header: "AI + Intune Detection Rules"
- Flow (5 steps):
  1) Gather installer intelligence
  2) Generate AI rule candidates
  3) Reliability scorecard
  4) Validation matrix
  5) Pilot rollout
- Right callouts:
  - Primary signals: MSI + version, registry patterns
  - Risky signal: file-exists only
- Bottom badges:
  - Fewer false negatives
  - Less package rework
  - Safer deployments
