# Drawy Whiteboard Spec — AI + SCCM Boundary Group Validation

## Goal
Create one whiteboard-style social visual summarizing a safe AI-assisted workflow for drafting and validating SCCM boundary group queries.

## Format
- 1:1 square (1080x1080)
- hand-drawn whiteboard style

## Core message
"AI can help desktop engineers draft SCCM boundary group validation queries faster, but the engineer must supply trusted schema, force assumptions into the open, and keep the first pass read-only."

## Layout
- Header: "AI + SCCM Boundary Group Queries"
- Flow (5 steps):
  1) Define the validation job
  2) Feed trusted schema or query skeleton
  3) Force assumptions to be labeled
  4) Keep first pass read-only
  5) Validate against known location
- Right callouts:
  - Don’t let AI guess ConfigMgr schema
  - Reporting first, production changes later
- Bottom badges:
  - Faster query drafting
  - Safer validation reports
  - Fewer WAN surprises
