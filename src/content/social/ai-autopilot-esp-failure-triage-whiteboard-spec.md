# Drawy Whiteboard Spec — AI Autopilot ESP Failure Triage

## Goal
Create one whiteboard-style social visual summarizing a practical AI-assisted workflow for Windows Autopilot Enrollment Status Page troubleshooting.

## Format
- 1:1 square (1080x1080)
- hand-drawn whiteboard style

## Core message
"AI helps desktop engineers triage Autopilot ESP failures faster when the evidence packet is structured and validated."

## Layout
- Header: "AI + Autopilot ESP Triage"
- Flow (5 steps):
  1) Capture ESP phase
  2) Collect blocking evidence
  3) Ask AI for ranked causes
  4) Validate in Intune and logs
  5) Escalate with clean summary
- Right callouts:
  - Device prep vs device setup vs account setup
  - Check detection rules before blaming ESP
- Bottom badges:
  - Faster root-cause narrowing
  - Better escalations
  - Fewer blind changes
