# Drawy Whiteboard Spec — AI + Intune App Supersedence Review

## Goal
Create one whiteboard-style social visual summarizing a safe AI-assisted review workflow for Intune app supersedence changes.

## Format
- 1:1 square (1080x1080)
- hand-drawn whiteboard style

## Core message
"AI can accelerate Intune app supersedence review, but engineers must validate uninstall behavior, detection logic, dependencies, blast radius, and rollback before production."

## Layout
- Header: "AI + Intune Supersedence Review"
- Flow (7 steps):
  1) Collect old app facts
  2) Collect new app facts
  3) Verify uninstall path
  4) Check detection collisions
  5) Review dependencies and reboot impact
  6) Limit assignment blast radius
  7) Define rollback before pilot
- Right callouts:
  - Don’t trust file-exists-only detection
  - No rollback = no-go
- Bottom badges:
  - Safer upgrades
  - Fewer pilot surprises
  - Better rollback readiness
