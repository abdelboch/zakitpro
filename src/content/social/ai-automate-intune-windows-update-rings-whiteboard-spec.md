# Drawy Whiteboard Spec — AI + Intune Windows Update Rings

## Goal
Create one whiteboard-style social visual summarizing a safe AI-assisted workflow for managing Microsoft Intune Windows Update Rings.

## Format
- 1:1 square (1080x1080)
- hand-drawn whiteboard style

## Core message
"AI can accelerate Intune update ring automation, but engineers must protect ring scope, blackout dates, and existing settings before any Graph change."

## Layout
- Header: "AI + Intune Update Rings"
- Flow (5 steps):
  1) Define pilot / broad / VIP scope
  2) Protect blackout dates and exclusions
  3) Generate Graph script with explicit settings
  4) Review payload before PATCH/POST
  5) Validate read-back in Intune
- Right callouts:
  - Preserve existing values
  - Never let AI guess production defaults
- Bottom badges:
  - Faster ring changes
  - Safer patch cadence
  - Fewer production surprises
