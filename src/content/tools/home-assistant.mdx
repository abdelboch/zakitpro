## Why Home Assistant?

For home automation, nothing beats Home Assistant's ecosystem. It connects to 2000+ devices and services, runs locally (no cloud required), and has the most active community in the space. For IT pros, it's also a great way to learn YAML, APIs, and IoT architecture.

## Key Features

- **2000+ Integrations** — Philips Hue, Zigbee, Z-Wave, MQTT, Wi-Fi, Bluetooth
- **Local Control** — No cloud dependency, works even when internet is down
- **Automations** — Powerful YAML or visual automation editor
- **Dashboards** — Custom Lovelace UI for monitoring and control
- **Energy Monitoring** — Track power consumption per device

## Self-Hosted Setup

**Recommended:** Home Assistant OS on a dedicated device (Raspberry Pi 4, Intel NUC, or ODROID)

```bash
# Docker alternative (limited functionality)
docker run -d \
  --name homeassistant \
  --privileged \
  --restart=unless-stopped \
  -e TZ=America/Denver \
  -v /path/to/config:/config \
  --network=host \
  ghcr.io/home-assistant/home-assistant:stable
```
