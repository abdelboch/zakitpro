## Why Uptime Kuma?

Most uptime monitors are ugly and overcomplicated. Uptime Kuma fixes both problems — it's beautiful AND it just works. I've replaced PRTG's basic monitoring with this for small teams.

## Key Features

- **50+ Monitor Types** — HTTP(s), TCP, Ping, DNS, Docker, gRPC, and more
- **Beautiful Dashboard** — Status pages your non-technical stakeholders will actually look at
- **Multi-Notification Channels** — Slack, Discord, Telegram, Email, Webhook, 90+ integrations
- **Certificate Monitoring** — Get alerted before SSL certs expire
- **Multi-User** — Team access with different permission levels

## Self-Hosted Setup

```bash
docker run -d \
  --restart=always \
  -p 3001:3001 \
  -v uptime-kuma:/app/data \
  --name uptime-kuma \
  louislam/uptime-kuma:1
```

Access at `http://your-server:3001` and create your admin account.
