## Why Nextcloud?

If you're tired of Google Drive and Microsoft 365 subscriptions, Nextcloud is the full-featured alternative you host yourself. We run it for our entire IT department.

## Key Features

- **File Sync & Share** — Full Dropbox replacement with desktop/mobile apps
- **Office Collaboration** — Built-in Collabora/OnlyOffice integration
- **Calendar & Contacts** — CalDAV/CardDAV with shared team calendars
- **Talk** — Video conferencing and chat
- **500+ Apps** — Extend with integrations for everything from LDAP to GitHub

## Self-Hosted Setup

The easiest way is with Docker:

```bash
docker run -d \
  --name nextcloud \
  -p 8080:80 \
  -v nextcloud_data:/var/www/html \
  nextcloud
```

For production, use their AIO (All-in-One) appliance or deploy on a Synology/QNAP NAS.

## Who It's For

- Teams replacing Google Workspace / Microsoft 365
- IT departments needing on-prem file sharing
- Privacy-conscious individuals
- MSPs offering hosted collaboration to clients
